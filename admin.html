🔑 How to get your GitHub Token (do this once):
  1. Go to github.com/settings/tokens/new
  2. Set a name like wagd-admin
  3. Set expiration (e.g. No expiration or 1 year)
  4. Under Scopes, tick repo (full control of repos)
  5. Click Generate token and copy it — you only see it once!
  6. Paste it below along with your GitHub username and repo name

Your token is stored only in your browser — never sent anywhere except GitHub's official API.
